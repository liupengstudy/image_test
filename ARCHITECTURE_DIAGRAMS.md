## 图像生成流程图

```
  客户端                                        服务器
    |                                            |
    |--- 1. 发送提示词、宽高比和参考图像 --------->|
    |                                            |
    |                                            |--- 2. 调用QWQ模型优化提示词 ----> QWQ模型
    |                                            |                                    |
    |                                            |<--- 3. 返回优化后的提示词 ----------+
    |                                            |
    |                                            |--- 4. 根据宽高比确定图像尺寸
    |                                            |
    |                                            |--- 5. 调用通义万相生成图像 -----> 通义万相
    |                                            |                                    |
    |                                            |<--- 6. 返回生成的图像 ------------+
    |                                            |
    |                                            |--- 7. 保存生成历史
    |                                            |
    |<-- 8. 返回生成的图像和优化后的提示词 -------|
    |                                            |
    |--- 9. 显示生成结果 ------------------+     |
    |                                      |     |
    |<-------------------------------------+     |
```

## 错误处理流程图

```
  客户端                                        服务器
    |                                            |
    |--- 1. 发送API请求 ------------------------->|
    |                                            |
    |                                            |--- 2. 请求处理
    |                                            |
    |                 正常情况                     |
    |                                            |
    |<-- 3a. 返回成功响应 -----------------------|
    |                                            |
    |--- 4a. 处理响应并更新UI ---------|          |
    |                                 |          |
    |<--------------------------------+          |
    |                                            |
    |                 错误情况                     |
    |                                            |
    |<-- 3b. 返回错误响应 -----------------------|
    |                                            |
    |--- 5b. 检查错误类型                         |
    |                                            |
    |--- 6b. 基于错误类型执行操作:                 |
    |        - 显示用户友好错误消息                |
    |        - 记录错误日志                       |
    |        - 应用错误恢复策略 --------|          |
    |                                 |          |
    |<--------------------------------+          |
    |                                            |
    |--- 7b. 如果适用，自动重试 ----------------->|
    |                                            |
```
